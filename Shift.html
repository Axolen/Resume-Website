Asteroid//Shift

Asteroid//Shift is a fast-paced 1-4 player FPS that takes place in space. This is a quick round based game were the gravity can switch at any moment! All of your projectiles bounce off the walls, increasing in speed each time it happens. Don't get hit by your own shot when it comes back at you!

This project has been so fun to work on. I've built all the features in the game, as well as the .net framework! Everything from Gravity Flipping, Game Physics, level design & sounds. I've learned so much from making this project.

Everything you see below is in ALPHA and I am very proud of everything I've done! Just know that it is a work in progress and will continue to be improved as I learn more!
